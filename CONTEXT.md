# STILL ALIVE (멈춰야 산다)

NAN 2026 (NHN Game × AI Hackathon) 사전 과제로 제출하는 서바이벌 액션. 심사자가 링크 클릭 한 번으로 브라우저에서 플레이할 수 있어야 한다.

## Language

**사전 과제**:
2026-08-10까지 제출하는 예선 결과물. 게임 + 제출물 5종 전체를 가리킨다.
_Avoid_: 본선 작품, 최종 제출

**본행사**:
사전 과제 통과 팀만 참여하는 48시간 오프라인 해커톤(9/4~6). 주제는 현장에서 공개된다.
_Avoid_: 본선 (모호함 — 사전 과제 심사와 혼동됨)

**AI 활용 기술 문서**:
AI를 어떻게 지휘해 게임을 만들었는지 서술하는 제출물. 프롬프트, 도구, 에셋 출처를 담는다. 게임과 동급의 심사 대상.
_Avoid_: 개발 후기, README

## 게임 시스템

**불릿타임**:
집중 게이지를 태워 세상을 늦추는 능동 능력 (Shift 홀드). 플레이어 본인은 실시간 속도를 유지한다. 세상이 저절로 느려지는 일은 없다 — 시간은 항상 자원의 대가다.
_Avoid_: 슬로우 모드, 시간 정지, (구) 시간 흐름 규칙

**콤보**:
킬 연속으로 오르는 점수 배율. 실시간으로 감소한다 — 불릿타임으로도 보존되지 않는다.
_Avoid_: 킬 스트릭

**집중 게이지**:
완전히 멈춰 서 있는 동안 실시간으로 차는 자원. 불릿타임과 대시가 소모한다. 충전 중에도 세상은 정상 속도로 다가온다 — 충전은 위험한 투자다.
_Avoid_: 마나, 스태미나

**대시**:
집중 게이지를 소모하는 순간 돌진. 무적 프레임이 붙는다.

**추적자**:
플레이어를 직선 추적하는 기본 적 (빨간 원). 0초부터 등장.

**예측자**:
플레이어의 이동 방향 앞 지점을 조준하는 적 (주황 삼각형). 경로 탐색이 아니라 리드 조준이다. 45초부터.
_Avoid_: 지능형 적, AI 적

**탱커**:
느리지만 단단한 벽 역할 적 (암적색 대형 원). 90초부터.

**돌진자**:
사거리 안에서 점멸 경고 후 직선 돌진하는 적 (보라 사각형). 경고가 있어 죽음이 항상 내 탓이다. 150초부터.
_Avoid_: 차저

**슬라임**:
같은 단계끼리 서로를 찾아가 합체하는 적 (청록 원). 소→중(2합체)→대(4합체 상당) 3단계로, 클수록 강해진다. 60초부터.
_Avoid_: 합체 적, 엘리트 (구 시스템 용어)

**대형 슬라임**:
슬라임 최종 단계. 주기적으로 4방향 탄을 쏘는 유일한 원거리 일반 적. 합체를 방치한 대가다. 동시 2마리 상한.
